Features:

Stuff Roblokis Can do
Feature Description
Custom Robux/Name you can change the text of Robux/Username on the top right corner to anything you want.
Quick Join Buttons on home and discover pages there will be a join button under every game.
Desktop App on the left navigator there is a new button that opens Roblox's App for Desktop.
New Design On some pages it will show you the site with a new design.
Design Editor On the Designer tab, you can create your own design.
Show Hidden Badges On the game details page it will show you badges that are disabled/hidden.
New Style On the game servers page, Servers will have a new style.
Small Server Section On the game details page on servers there will be a new section called "Some Small Servers" there will be servers with few members in it.
Copy Servers Link On the game details pages on servers it will show a link button next to join buttons.
That button will give you server's link and you can paste it as url to join the server.
Automatic Load More On the game details pages on servers it will click load more for private servers.
Available Servers Only On the game details pages on servers it will ONLY show private servers with a join button.
Rename Join button On the game details pages on servers it change text of friends servers join button to either "Join Public Server" or "Join Private Server".
Server Page Navigator On the game details pages on servers it will show navigation buttons under Load More button on Public Servers Section.
Quick Friend Remover On the friends page on friends tap it will show a button to remove friends faster.
Last Player Status On any profile page, it will show you players last activity.
Old Roblox Status On any profile page, it will show you the old players status by roblox.
Cencel Friend Request On any profile page; if you sent a friend request to someone, you can cancel it bu the same button.
